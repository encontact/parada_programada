# parada_programada
Site de aviso para momentos de parada programada podermos redirecionar para este site.

## Como usar

1. Hospede o arquivo `index.html` em um servidor web ou serviço de hospedagem estática
2. Configure o redirecionamento do seu domínio para este site durante períodos de manutenção
3. O site exibe automaticamente as informações sobre a parada programada (23h00 às 03h00)

## Recursos

- Logo do enContact (com fallback para texto caso a imagem não carregue)
- Informações claras sobre o período de manutenção
- Instruções de contato para suporte
- Design responsivo que funciona em dispositivos móveis e desktop
